####################
燧原 GCU 芯片
####################

燧原 S60 是面向数据中心大规模部署的新一代人工智能推理加速卡，满足大语言模型、搜广推及传统模型的需求，具有模型覆盖面广、易用性强、易迁移易部署等特点，可广泛应用于图像及文本生成等应用、搜索与推荐、文本、图像及语音识别等主流推理场景。更多燧原 GCU 芯片详情及技术指标请 `点击这里 <https://www.enflame-tech.com/>`_ 。
